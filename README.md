# 📖 Overview

**Verdict** is an AI-powered fashion purchase decision support platform designed to help users make smarter clothing purchase decisions.

Instead of simply telling users whether an outfit "looks good," Verdict evaluates whether a garment is appropriate for a specific occasion and explains every recommendation using transparent, structured reasoning.

The platform combines **OpenAI GPT-4.1 Vision**, deterministic application logic, runtime validation, and a personalized **Style DNA** profile to generate trustworthy fashion guidance.

---

# ✨ Why Verdict?

Shopping online often leads to uncertainty:

- ❓ Will this outfit suit me?
- ❓ Is it appropriate for the occasion?
- ❓ Does it match my personal style?
- ❓ Is there a better alternative available?

Most fashion AI tools provide only subjective opinions or numerical ratings.

**Verdict** takes a different approach by delivering explainable recommendations backed by structured evaluation criteria.

---

# 🚀 Key Features

## 🤖 AI Outfit Analysis

Analyze outfits using OpenAI GPT-4.1 Vision.

The AI evaluates:

- Occasion Fit
- Color Harmony
- Formality
- Seasonality
- Style Consistency
- Style Preference Match

---

## 🎯 Deterministic Verdict Engine

Unlike many AI applications, the final Verdict is **not generated by AI**.

Instead, the application calculates the final result using deterministic business rules, making every recommendation:

- Transparent
- Reproducible
- Explainable
- Consistent

---

## 🧬 AI Style DNA

Users create a persistent Style DNA profile describing:

- Preferred aesthetics
- Favorite colors
- Fashion personality
- Comfort preferences
- Lifestyle
- Preferred fits

Style DNA personalizes future recommendations without storing original photos.

---

## 🛍️ Shopping Advisor

After analysis, Verdict recommends better alternatives tailored to the user's Style DNA.

Recommendations include:

- Personalized explanations
- Match quality
- Expected outcome
- Direct retail search links

Supported retailers:

- Myntra
- Amazon
- AJIO

---

## 🔒 Privacy First

Verdict is designed with privacy as a core principle.

- Original images are not permanently stored.
- Style DNA is stored locally.
- AI recommendations remain explainable.
- Users maintain control over their personal style profile.
