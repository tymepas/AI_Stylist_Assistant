# 📖 Overview

**Verdict** is an AI-powered fashion purchase decision support platform designed to help users make smarter clothing purchase decisions.

Instead of simply telling users whether an outfit "looks good," Verdict evaluates whether a garment is appropriate for a specific occasion and explains every recommendation using transparent, structured reasoning.

The platform combines **OpenAI GPT-4.1 Vision**, deterministic application logic, runtime validation, and a personalized **Style DNA** profile to generate trustworthy fashion guidance.

---

# ✨ Why Verdict?

Shopping online often leads to uncertainty:

- ❓ Will this outfit suit me?
- ❓ Is it appropriate for the occasion?
- ❓ Does it match my personal style?
- ❓ Is there a better alternative available?

Most fashion AI tools provide only subjective opinions or numerical ratings.

**Verdict** takes a different approach by delivering explainable recommendations backed by structured evaluation criteria.

Unlike many AI applications, the final Verdict is **not generated by AI**.

## 🔒 Privacy First

Verdict is designed with privacy as a core principle.

# 🧬 Style DNA

Style DNA is the personalization engine behind Verdict.

Instead of making recommendations using only the uploaded images, Verdict builds a reusable profile that represents the user's fashion preferences.

Style DNA captures:

- Preferred aesthetics
- Favorite colors
- Clothing preferences
- Lifestyle
- Comfort level
- Fashion goals
- Occasion preferences

This allows recommendations to remain consistent across future analyses.

---

## Why Style DNA?

Traditional fashion AI evaluates only the current image.

Verdict evaluates:

Current Outfit

+

Current Occasion

+

Personal Style DNA

This creates recommendations that feel significantly more personalized.

# ⚙️ Deterministic Verdict Engine

One of Verdict's biggest design decisions is separating AI reasoning from business logic.

The AI **never decides the final verdict.**

Instead:

AI provides structured observations.

↓

Application validates the response.

↓

Application calculates the final verdict.

This guarantees:

- Consistency
- Explainability
- Transparency
- Repeatability

# 🎯 AI Responsibilities vs Application Responsibilities

A major design goal was keeping AI responsible only for perception and reasoning while the application remains responsible for decision making.

| AI Responsibilities | Application Responsibilities |
|---------------------|------------------------------|
| Analyze uploaded images | Validate AI response |
| Detect colors | Calculate Verdict |
| Evaluate formality | Apply business rules |
| Evaluate occasion fit | Compute weighted score |
| Generate explanations | Shopping recommendations |
| Suggest improvements | Render final report |

This separation improves reliability, consistency, and explainability.
